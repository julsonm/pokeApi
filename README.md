# pokeApi
Pokémon Search Page

Este repositorio contiene una aplicación web simple para buscar información sobre Pokémon utilizando la PokeAPI. Además, se ha integrado la funcionalidad de activación por voz mediante la biblioteca Artyom.

Ejecución de la Aplicación:
1.Clonar el Repositorio:
git clone https://github.com/tu-usuario/pokemon-search-page.git
2.Abrir el Proyecto:
Navega hacia el directorio del proyecto clonado.
3.Ejecutar la Aplicación:
Abre el archivo index.html en tu navegador web preferido.
-Funcionalidades Implementadas
Búsqueda de Pokémon: Ingresa el nombre de un Pokémon en el campo de búsqueda y presiona el botón "Buscar" para obtener información detallada sobre él, incluida su imagen, nombre, número de Pokédex, habilidades, tipo y movimientos.
Activación por Voz: Presiona el botón "Activar Voz" para activar el modo de escucha por voz. Puedes pronunciar los nombres de los pokemones para recibir respuestas auditivas correspondientes. Además, puedes hablar el nombre de un Pokémon para buscarlo automáticamente.
Dependencias
Bootstrap: Se utiliza Bootstrap para el diseño y la maquetación de la página web.
jQuery: Se utiliza jQuery para facilitar la manipulación del DOM y hacer solicitudes AJAX.
PokeAPI: Se utiliza la PokeAPI para obtener datos sobre Pokémon.
Artyom: Se utiliza la biblioteca Artyom para la activación por voz y el reconocimiento de comandos de voz.
